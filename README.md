# IntellMeet Enterprise Platform

AI-Powered Enterprise Meeting & Collaboration Platform featuring secure multi-peer WebRTC video streaming, Socket.io real-time network orchestration, token-based Express backend authentication, and automated AI post-meeting insights with dynamic PDF transcript generation. Built with React, Next.js, TypeScript, Node.js, and Tailwind CSS.

## 🚀 Live Demo
Check out the live application here: [IntellMeet Live Deployment](https://intell-meet-enterprise-platform.vercel.app)

---

## ✨ Features

- **Real-Time Video/Audio:** High-quality multi-peer video streaming powered by WebRTC.
- **Network Orchestration:** Real-time event handling, chat, and room management using Socket.io.
- **Secure Authentication:** Robust token-based user authentication on the Express backend.
- **AI-Powered Post-Meeting Insights:** Automatically generates post-meeting summaries, action items, and insights.
- **Dynamic PDF Transcripts:** Downloadable structured PDF documents of your meeting transcripts and summaries.
- **Modern UI/UX:** Fully responsive, modern Dashboard built using React, Next.js, and Tailwind CSS.

---

## 🛠️ Tech Stack

**Frontend:**
- React.js / Next.js (App Router)
- TypeScript
- Tailwind CSS
- WebRTC & Socket.io-client

**Backend:**
- Node.js
- Express.js
- Socket.io
- Supabase (Database / Authentication / Storage)
